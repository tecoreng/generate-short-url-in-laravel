# Laravel -  Generate Short URL (Without Using Composer Package)

![laravel generate short URL](https://imgtr.ee/images/2023/06/07/pxaZq.png)


# Installation and running

#### 1. Install Laravel
first of all we need to get fresh Laravel application using bellow command, So open your terminal OR command prompt and run bellow command.

```bash

composer create-project --prefer-dist laravel/laravel blog

```

#### 2. Run Migration
Create migration for "short_links" table using Laravel php artisan command, so first fire bellow command:

```bash

php artisan migrate

```
Then set id,code and link filed in migration table.

#### 3. Run application

rename .env.example file to .env and set your configuration.

```bash

php artisan serve

```

Now you can open bellow URL on your browser:


```bash

http://localhost:8000

```
